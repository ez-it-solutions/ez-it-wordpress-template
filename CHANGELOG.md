# Changelog

All notable changes to the Ez IT WordPress Plugin Template will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-07

### Added
- Initial release of Ez IT WordPress Plugin Template
- Modern tabbed admin interface with AJAX navigation
- Dark and light theme support with persistent storage
- Interactive theme selector with visual preview cards
- Advanced settings tab with toggle switches
- Beautiful UI components (cards, stats, toggles, buttons)
- Comprehensive design system in BRANDING.md
- Complete setup guide for customization
- Responsive design for all screen sizes
- Security best practices (nonces, capability checks, sanitization)
- Loading states and smooth transitions
- Sidebar panels for help and information
- Full WordPress admin notice suppression
- Clean, well-documented code structure

### Features
- **Tabbed Navigation**: Dashboard, Settings, and About tabs
- **Theme Toggle**: Instant switching between dark and light modes
- **Settings Management**: Cache, debug mode, auto-cleanup, token expiry
- **Stats Cards**: Eye-catching cards with hover effects
- **Toggle Switches**: Beautiful on/off switches matching AeroCheckout style
- **Color System**: Lime green (#a3e635) for dark mode, green (#16a34a) for light mode
- **Typography**: Well-defined font system with consistent sizing
- **Spacing Scale**: Consistent margins and padding throughout
- **Animations**: Smooth transitions and hover effects

### Documentation
- README.md - Complete overview and quick start guide
- SETUP-GUIDE.md - Step-by-step customization instructions
- BRANDING.md - Full design system documentation
- CHANGELOG.md - This file

### Technical Details
- WordPress 6.0+ compatible
- PHP 7.4+ required
- MySQL 5.7+ required
- Follows WordPress coding standards
- AJAX-powered tab loading
- Modular file structure
- Easy to extend and customize
